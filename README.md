# Month 2
# Customer Churn Analysis 

## Objective

Analyze customer churn behavior and identify key factors influencing customer retention.

##  Tools & Technologies

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-learn

##  Dataset

Telecom Customer Churn Dataset

##  Key Steps

* Data Cleaning (handling missing values, fixing data types)
* Feature Engineering (tenure groups, average spend)
* Encoding categorical variables
* Exploratory Data Analysis (EDA)

## Visualizations

* Churn distribution (countplot)
* Correlation heatmap
* Monthly charges vs churn (boxplot)

##  Key Insights

* Customers with month-to-month contracts have higher churn
* Higher monthly charges increase churn probability
* Long-term customers are less likely to churn

## Files

* customer_churn_analysis.ipynb
* raw_data.csv

##  Outcome

Built a complete data preprocessing and EDA pipeline and applied a basic machine learning model to predict churn.
