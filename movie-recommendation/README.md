# Movie Recommendation System 🎬

##  Objective

Build a content-based movie recommender using text features like genres, cast, keywords, and descriptions.

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* NLP (CountVectorizer)
* Cosine Similarity

##  How It Works

* Combined multiple features into a single "tags" column
* Converted text into vectors using CountVectorizer
* Calculated similarity using cosine similarity
* Recommended top 5 similar movies

##  Features

* Recommend movies based on content similarity
* Uses NLP techniques for feature extraction

##  Files

* notebook.ipynb
* movies.pkl
* similarity.pkl

##  Dataset

TMDB 5000 Movie Dataset
Download from Kaggle: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
