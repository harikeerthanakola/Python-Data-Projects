# Netflix Titles Analysis 

##  Objective

Analyze Netflix content to identify trends in movies, TV shows, genres, and countries.

##  Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Steps

* Data Cleaning (handling missing values, date conversion)
* Feature Engineering (release year, decade, content type)
* Exploratory Data Analysis (EDA)

##  Visualizations

* Movies vs TV Shows distribution
* Content growth over years
* Top genres
* Country-wise content distribution

##  Key Insights

* Content increased rapidly after 2015
* Movies dominate over TV shows
* USA and India produce most content
